Using GitHub

GitHub is our main repository for ALL information related to OSTE. Knowing how to use GitHub is very important in collaborating with the OSTE network and providing new or edited documentation to expand Technology Education. If you are not familiar with using GitHub please go ahead and follow my quick tutorial. I have also included a GitFlow PDF on branching techniques as well.

Tutorial

1. First Download the GitHub desktop client

2. After installing, select the 'gears' in order to configure GitHub for Desktop

3. Configure GitHub with your GitHub account and display information while also selecting where to download cloned repos.

4. Locate the main OSTE repository here at https://github.com/avertucci/Open-Source-TechEd.

5. Select download to clone the repository to your PC or Mac.



6. Once cloned to your desktop, you can BRANCH off of the master branch and grab everything needed to collaborate. This will create your own branch that you can then work in and merge back in with a PULL REQUEST.



6. Be sure to save your changes in your commits before creating a Pull Request and be as descriptive as possible.
